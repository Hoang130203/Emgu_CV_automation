## Run the Application

Open a terminal in this directory and run:

```bash
dotnet run --project src/UI/GameAutomation.UI.WPF/GameAutomation.UI.WPF.csproj
```

Or navigate to the WPF project folder first:

```bash
cd src/UI/GameAutomation.UI.WPF
dotnet run
```
